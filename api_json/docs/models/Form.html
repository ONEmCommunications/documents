Form

  • type : String
    The type of a form is always form
  • body : List
    Sequence of components used to acquire the pieces of data needed from user
  • method : String
    The HTTP method used to send the form data
    Enum:
    GET
    POST
    PUT
    PATCH
    DELETE
    HEAD
    OPTIONS
    TRACE
  • path : String
    The path used to send the form data
  • header : String
    The header of the form. It can be overwritten by each body component
  • footer : String
    The footer of the form. It can be overwritten by each body component
  • meta :
    Contains configuration flags
